# Contributing to moddef-ts

Thanks for your interest. This page covers how contributions are licensed and
signed off.

## No CLA

There is no Contributor License Agreement. You keep the copyright to your
contribution and license it to the project under Apache-2.0, the same license
as the rest of this repository.

## Developer Certificate of Origin

Every commit must carry a `Signed-off-by` trailer certifying that you wrote the
change or have the right to submit it under the project's license. This is the
[Developer Certificate of Origin](https://developercertificate.org/) 1.1. Add
the trailer by committing with `-s`:

```bash
git commit -s -m "Your message"
```

Use your real name and an email you can be reached at. A DCO check runs on each
pull request.

## SPDX headers

New source files start with an SPDX header, using the comment syntax of the
language:

```
// SPDX-License-Identifier: Apache-2.0
```
